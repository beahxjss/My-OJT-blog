My 6-Week Journey

Each week brought new challenges, learning opportunities, and memorable experiences.

1
January 8-12, 2025

Orientation & Environment Setup

The first week was all about getting familiar with the company culture and development environment. I was introduced to the team, learned about ongoing projects, and spent time setting up my development workspace. The onboarding process was comprehensive, covering company policies, security protocols, and development best practices. I also attended several team meetings to understand the current sprint goals and project timelines.

Onboarding Team Introduction Environment Setup
2
January 15-19, 2025

Learning the Tech Stack

Learning!
} Code Learning Session

This week focused on diving deep into the company's technology stack. I spent time learning React.js, Node.js, and PostgreSQL - technologies that were new to me. My mentor assigned several small coding exercises to help me get comfortable with the codebase structure and coding standards. I also participated in code reviews to understand how the team maintains code quality and collaborative development practices.

React.js Node.js Code Reviews
3
January 22-26, 2025

First Real Project Assignment

My Dashboard
); Component Code

I received my first substantial project assignment - developing a user dashboard component for the company's main application. This was both exciting and nerve-wracking as it would be used by real customers. I worked closely with the UI/UX designer to understand the requirements and began implementing the component using React. The challenge was ensuring responsive design and accessibility compliance while maintaining the existing design system.

Dashboard Development UI/UX Collaboration Responsive Design
4
January 29 - February 2, 2025

Backend Integration & API Development

This week shifted focus to backend development. I learned how to create RESTful APIs using Node.js and Express, and integrated my frontend component with the backend services. Database design and optimization became a key learning area as I worked with PostgreSQL. I also gained experience with API testing using Postman and learned about proper error handling and validation techniques.

REST APIs Database Design API Testing
5
February 5-9, 2025

Testing & Quality Assurance

Quality assurance took center stage this week. I learned about different testing methodologies including unit testing with Jest, integration testing, and end-to-end testing with Cypress. Writing comprehensive tests for my dashboard component helped me understand the importance of maintainable code. I also participated in bug fixing sessions and learned about debugging techniques and performance optimization strategies.

Unit Testing Jest & Cypress Bug Fixing
6
February 12-16, 2025

Project Deployment & Reflection

The final week was dedicated to deploying my completed dashboard component to the staging environment and preparing for production release. I learned about CI/CD pipelines, Docker containerization, and deployment best practices. The week culminated in presenting my work to the entire development team and receiving feedback. It was incredibly fulfilling to see my code being used in a real production environment and knowing that real users would benefit from my work.

Deployment CI/CD Presentation
